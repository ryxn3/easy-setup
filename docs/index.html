Set up a fresh Windows PC in a few clicks.

Easy Setup is a small native app that installs the apps you want, flips the Windows settings you always change anyway, and cleans up the rest — all from one dark, Fluent-styled window.

How it works

STEP 1

Check what you want

Search or browse by category, and flip any settings you like along the way.

STEP 2

Click install

Each app installs via winget, always the latest version, with a live log as it runs.

STEP 3

Done

A fresh PC, set up your way, in the time it takes to make a coffee.

Build it yourself

Single C++ file, Win32 + Direct2D, no runtime dependencies. Needs Visual Studio with the Desktop C++ workload.

git clone https://github.com/ryxn3/easy-setup.git cd easy-setup && Build.bat
Windows 10/11 · winget required for installs · no telemetry
