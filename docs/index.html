Installers

Search or check off what you need — Steam, Chrome, VS Code, Docker, 63 apps in all — and install the latest version of everything in one pass via winget.

Windows Settings

Flip the toggles you reset on every fresh install — taskbar on every monitor, End Task in the right-click menu, dark mode, developer mode — straight from the registry, no digging through Settings.

Misc

Startup app management, Disk Cleanup, Device Manager, SFC scans, restore points and 30-odd other one-click shortcuts you'd otherwise have to remember the .cpl name for.

How it works

STEP 1

Check what you want

Search or browse by category, and flip any settings you like along the way.

STEP 2

Click install

Each app installs via winget, always the latest version, with a live log as it runs.

STEP 3

Done

A fresh PC, set up your way, in the time it takes to make a coffee.

Build it yourself

Single C++ file, Win32 + Direct2D, no runtime dependencies. Needs Visual Studio with the Desktop C++ workload.

git clone https://github.com/ryxn3/easy-setup.git cd easy-setup && Build.bat
